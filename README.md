# Лудо (Ludo) — GUI (Swing)

Java-приложение с графическим интерфейсом (Swing), реализующее игру **Лудо** по правилам из задания.

Что можно показать на видео:
- окно с полем (кольцо + базы + дорожки к дому);
- кнопка **«Бросить кубик»**;
- выбор фишки кликом (доступные фишки подсвечены)
- перемещения по правилам (блоки/безопасные клетки/срубание/бонусы/доп. ход на 6).

## Запуск
### Через Maven

```fish
mvn test
mvn -q exec:java
```
